Color System

OKLCH Color Palette

Perceptually uniform colors with predictable lightness and wider gamut support for modern displays.

Why OKLCH?

Modern color space with superior perceptual properties

Perceptual Uniformity

Equal numeric changes produce equal visual differences, unlike RGB/HSL.

Predictable Lightness

The L value directly maps to perceived brightness, making calculations reliable.

Wider Gamut

Supports P3 color space for modern displays with richer, more vibrant colors.

Better Gradients

Interpolation doesn't pass through muddy colors, creating smoother transitions.
