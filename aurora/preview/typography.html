Readability Guidelines

Principles for comfortable, scannable text

✅ Optimal Line Length

This paragraph demonstrates optimal line length at 65 characters. Research shows that 50-75 characters per line is ideal for comfortable reading. This helps the eye track from one line to the next without getting lost or fatigued.

Max width: 65ch (characters) • ~672px at 16px

❌ Too Wide

This paragraph is too wide and demonstrates poor readability. When lines are too long, the eye has difficulty tracking from the end of one line to the beginning of the next. This creates fatigue and reduces comprehension. Studies show that extremely long lines significantly decrease reading speed and comfort.

No max-width constraint • Causes eye strain

Line Height: 1.5 for Body

Body text uses 1.5 line height (24px at 16px font size). This generous spacing improves readability by giving each line room to breathe. The vertical rhythm helps guide the eye naturally from line to line.

1.5 line height = comfortable reading

Line Height: 1.15 for Headings

Headings use tighter 1.15 line height for visual impact. Larger text doesn't need as much vertical space, and tighter leading creates a more cohesive, powerful appearance.

1.15 line height = tight, impactful

Typography Best Practices

Do's and don'ts for optimal readability

✅ Do

  • • Use 16px minimum for body text
  • • Maintain 1.5 line height for readability
  • • Limit line length to 50-75 characters
  • • Use weight 600 for headings
  • • Apply negative tracking to large text
  • • Ensure 4.5:1 contrast ratio minimum
  • • Use proper heading hierarchy (H1→H2→H3)

❌ Don't

  • • Go below 14px for body text (12px absolute minimum)
  • • Use line height < 1.4 for paragraph text
  • • Create lines longer than 80 characters
  • • Mix too many font weights (stick to 400, 500, 600)
  • • Over-use uppercase text (hard to read)
  • • Use low contrast text for body content
  • • Skip heading levels (H1→H3 without H2)
